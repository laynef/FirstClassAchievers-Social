# FirstClassAchievers

This is a Social Web App

## To Install & Run Web

`npm i`<br>
`npm start` <br>

## To Install & Run Mobile

`cd native`<br>
`npm i`<br>

For iOS<br>
`react-native run-ios`<br>
For Android<br>
`react-native run-android`<br>
